// Requirements

Before you run a Ghost

  • NVIDIA H100/H200 with confidential computing, or AMD EPYC with SEV-SNP (capability probe will verify).
  • Solana wallet with GHST for staking and job routing (see the console for live pricing).
  • macOS 11+, Windows 10/11, or a recent Linux distro for AppImage / .deb / .rpm packages.
  • Unsigned alpha builds — macOS may require right-click → Open on first launch.
